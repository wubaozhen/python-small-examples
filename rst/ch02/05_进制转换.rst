5 十转二
--------

将\ ``十进制``\ 转换为\ ``二进制``

.. code:: python

   In [1]: bin(10)
   Out[1]: '0b1010'

6 十转八
--------

将\ ``十进制``\ 转换为\ ``八进制``

.. code:: python

   In [1]: oct(9)
   Out[1]: '0o11'

7 十转十六
----------

将\ ``十进制``\ 转换为\ ``十六进制``

.. code:: python

   In [1]: hex(15)
   Out[1]: '0xf'
