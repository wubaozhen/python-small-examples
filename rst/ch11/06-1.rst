
1 了解数据
----------

数据来自kaggle，共包括三个文件：

1. movies.dat

2. ratings.dat

3. users.dat

``movies.dat``\ 包括三个字段：['Movie ID', 'Movie Title', 'Genre']

使用pandas导入此文件：

.. code:: python

    import pandas as pd

    movies = pd.read_csv('./data/movietweetings/movies.dat', delimiter='::', engine='python', header=None, names = ['Movie ID', 'Movie Title', 'Genre'])
